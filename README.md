# pyてょん to python [refactor/js-to-ts]
日本語入力で変換確定後に誤変換「pyてょん」を「python」に変換するブラウザ拡張機能です<br>
**ソースをJavaScriptからTypeScriptに変更するためのブランチです。**
## 概要
JavaScriptを実装していたものをTypeScriptに変更します。<br>
今後はTypeScriptで作って行きたいと思っています。<br>
TypeScriptは初めてなので学びながら実装します。<br>
