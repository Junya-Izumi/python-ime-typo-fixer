# pyてょん to python
日本語入力で変換確定後に誤変換「pyてょん」を「python」に変換しましす<br>
間違えた時にちょっと楽になればと思って作りました
## インストール
1. このリポジトリの clone または dowonload する
2. Chrome で 'chrome://extensions' を開く
3. デベロッパーモードをONにする
4. 「パッケージ化した拡張機能を読み込む」を選択
5. このフォルダを選択
